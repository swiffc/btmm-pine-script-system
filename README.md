## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 14+ installed
- Git installed and configured  
- TradingView account
- Cursor IDE (recommended) or VS Code

### **Installation & Setup**
```bash
# Clone the repository
git clone https://github.com/swiffc/btmm-pine-script-system.git
cd btmm-pine-script-system

# Install dependencies and initialize system
npm run setup

# All changes automatically sync to GitHub!
```

### **🔄 Development Workflow (Automated)**
```bash
# System automatically organizes files and pushes to GitHub
npm run organize-files        # Professional file organization
npm run commit:auto          # Automated commit with validation
# GitHub push happens automatically per BTMM rules!

# Deploy to TradingView
npm run deploy <script>      # Deploy single script
npm run deploy-all          # Deploy all scripts

# System validation
npm run validate-all         # Full system health check
npm run status              # Current system status
```
